LGTM
===

[![GoDoc](https://godoc.org/github.com/ktr0731/lgtm?status.svg)](https://godoc.org/github.com/ktr0731/lgtm)

![](./lgtm.png)  

## Install

``` bash
$ go get github.com/ktr0731/lgtm/cmd/lgtm
```

## Usage

``` bash
$ lgtm -w example.gif
$ lgtm example.gif out.gif
```

## Supported type
- gif
- png
